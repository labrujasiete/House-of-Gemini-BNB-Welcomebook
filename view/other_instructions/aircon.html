AIRCON
Air conditioner instructions & maintenance
Instructions
  • Locate the wall-mounted controller in your room.
  • Press the power button to turn on the unit.
  • Use the mode button to choose "Cool" for cooling.
  • Use the temperature buttons to set your desired temperature.
  • Adjust fan speed using the fan speed button if needed.
  • Use timer and swing functions if available.
  • Check the display for your settings.
  • Turn off the unit when not needed to save energy.
  • If needed, consult the host or user manual for further assistance.
